Technologie JavaScript – TPs
Description
Bienvenue sur mon dépôt dédié au module Technologie JavaScript, actuellement en cours. Vous y trouverez mes TPs au fur et à mesure de mon avancement.
J'aborde ce module avec l'envie de vraiment comprendre JavaScript de l'intérieur — pas juste faire tourner du code, mais saisir pourquoi ça marche.
Ce que j'explore jusqu'ici :

Les bases solides du langage (variables, types, fonctions, boucles, conditions)
La manipulation du DOM et l'interaction avec les pages web
La gestion des événements utilisateur
Le JavaScript moderne avec les nouveautés ES6+
L'asynchrone avec les Promises et Async/Await

Et dans la suite du module :

React — construction d'interfaces dynamiques par composants
Node.js — JavaScript côté serveur
Et bien d'autres technologies de l'écosystème JS...
